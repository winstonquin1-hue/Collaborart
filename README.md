Collaborart has many uses.
The app itself is a multi user interactive way to make 
presentations, documents, videos and other uses.
It would even be possible to make some basic games because of
the interactive nature of the app.

Collaborart implements two technologies.
The first is Webkit which is the engine behind web browsers.
Webkit provides a way of connecting, in a very safe way, devices like
computers and smartphones to eachother.
The transactions are handled by Collaborart which acts as a manager for 
the data and dataflow between connected devices. 

One or many devices can connect to the same view and then work with it interactivly 
or just watch a presentation.
Its possible to display screencast, projectors etc. and this makes Collaborart
a very effective presentation maker. It does this in a unique way.
Presentations can be saved as a video using screen capture.

The app is a server and runs on a local device, for example a PC or an Android device.
Clients access the server through a web browser (web page) which is the interface to Collaborart.

Once the server is running, connected clients can collaborate on making a 
single document or presentation. 

This collaboration opens up many possibilities. 
Here are a few scenarios:

In an office a single document can have more that one person editing and contributing to it 
all in real time. This can be made into a presentation.

Since HTML can be placed into the document a whole lot of possibilities become available.
HTML 5 is packed with features to make web pages vibrant. They are focused on making highly
attractive web pages. With Collaborart a world of possibilities opens up.
By nature its able to data, pictures, videos or whatever from any available source.

Collaborart is intuitive and easy to use but the quickest way to learn how to use
this app is to watch a video or two available through the links provided.
