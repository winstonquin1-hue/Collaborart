Simple Stream Pump

This needs to be a picture from a URL because the fetch relies on http to fetch it.
If not the following error is raised... TypeError: Failed to fetch at STREAM_PUMP.html:21:3
Where 21 is the line with http request... fetch('./distribution/images/gridwork.png')
and 3 is where the 'f' in 'fetch' is found at pos 3.

The left tortoise is the original, the right one is created using a custom stream.
